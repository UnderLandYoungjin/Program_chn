## 📌 课程概览

| 项目 | 内容 |
|:----:|:----:|
| 课程名称 | 编程基础 |
| 授课科目 | 基础编程（C#） |
| 适用对象 | 人工智能软件专业 |

---

## 🎯 教学目标

- 掌握C#基本语法、数据类型、控制结构与数组，能独立编写控制台程序。
- 理解方法与类的设计思路，运用面向对象思维进行程序开发。
- 借助Windows Forms构建事件驱动的图形界面，并完成综合实战项目。

---

## 🗓️ 各周教学安排

| 周次 | 教学与实践内容 | 核心关键词 |
|:----:|:----------------|:----:|
| 第1周 | 搭建C#开发环境，理解程序基本结构 | Visual Studio, .NET, Hello World |
| 第2周 | 变量、数据类型与运算符实践 | int, string, bool, var |
| 第3周 | 输入输出与类型转换实践 | Console.WriteLine, Convert, 类型转换 |
| 第4周 | 条件判断与循环控制 | if, switch, for, while |
| 第5周 | 数组与字符串处理 | Array, string 方法 |
| 第6周 | 方法的定义与调用 | 定义, 参数, 返回值, 重载 |
| 第7周 | 类与面向对象编程基础 | OOP, 构造函数, 访问修饰符 |
| 第8周 | 继承、接口与集合类型的综合运用 | inheritance, interface, List, Dictionary |
| 第9周 | Windows Forms入门与事件处理机制 | Form, Button, TextBox, EventHandler |
| 第10周 | 图形界面综合项目实战 | 迷你项目 |

# 🗺️ 学习路线图

---

## 📚 第一学期 — 打牢基础

> 真正的实力，不是靠背语法背出来的，而是**动手做出来**的。

- 吃透C#基础语法 — 变量、条件、循环、类、GUI
- **完成迷你项目** — 计算器、成绩管理、小游戏……
- 哪怕项目再小，**坚持做完**才是最重要的习惯
- **考取资格证** — 用证书检验自己的成长
- 技能士 · 产业技师 · 工程师

---

## 🚀 第二学期 — 打造就业作品集

> 在第一学期的基础上，融入**AI · 嵌入式 · Web**技术，完成能拿出手的实战项目。

- AI Vision — Python, OpenCV, YOLO, MediaPipe
- 嵌入式开发 — AVR, IoT
- Web / 可视化面板 — Flask, FastAPI, React, Blazor
- **作品集项目** — 以就业为导向，打造真实可用的项目
- 求职准备 — 自我介绍撰写 · 技术面试备战

---

## 🐙 日常学习，记录在 GitHub 或社区

> 每一次提交，都是你成长最好的证明。

- 把课堂代码和迷你项目持续提交到仓库
- 养成写README的习惯 — 能讲清楚，才算真的懂了
- 求职时，一个GitHub主页的说服力，往往超过一份简历
